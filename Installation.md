# Installation #

## Ubuntu Hardy ##

Download package from [Download page](http://code.google.com/p/ditz-commander/downloads/list)

## Debian ##

Download package (with Ditz) from http://www.cgarbs.de/stuff/deb-repository.html#1

## Build from source ##

```
cd ditz-commander
dpkg-buildpackage -rfakeroot -us -uc
```
